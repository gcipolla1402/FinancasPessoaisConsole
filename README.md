## AppFinanceiro

Projeto simples em Java para controle financeiro pessoal via console.

## Funcionalidades

- Adicionar receitas e despesas
- Listar transações
- Visualizar saldo atual
- Exibir gráfico de despesas por categoria no console

## Como usar

1. Compile e execute o programa.
2. Use o menu para adicionar receitas/despesas e visualizar informações financeiras.
